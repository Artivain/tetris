Informations

Ceci est une version modifié (surtout simplifié) du Tetris disponible ici: HTML Tetris.

Comment jouer

• Clavier: W, S et D pour déplacer les pièces, W pour les tourner, P pour mettre le jeu en pause.
• Tactile: Glisser pour déplacer, appuyer pour tourner les pièces.
